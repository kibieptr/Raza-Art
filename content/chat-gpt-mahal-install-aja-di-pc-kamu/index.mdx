## Apa itu GPT?

Sebelum memulai, penting untuk memahami apa itu GPT. Menurut penjelasan yang ditemukan di website AWS, Generative Pre-trained Transformers (juga dikenal sebagai GPT) adalah keluarga model jaringan neural yang menggunakan arsitektur transformator dan merupakan kemajuan besar dalam kecerdasan buatan (AI) yang mendukung aplikasi AI generatif seperti ChatGPT. Model GPT memungkinkan aplikasi untuk membuat teks dan konten dengan tampilan yang mirip dengan buatan manusia.

## Apakah GPT dapat di install di lokal?

Karena model GPT dilatih menggunakan dataset yang besar dengan banyak pengetahuan, ukurannya biasanya sangat besar dan membutuhkan server dengan spesifikasi tinggi. Namun, berkat perkembangan AI, banyak model GPT yang dibuat lebih kecil dan dapat dijalankan pada komputer lokal dengan spesifikasi stand. Selain itu, LM Studio adalah salah satu dari banyak proyek open source yang saat ini dapat digunakan untuk menjalankan model ini.

Anda dapat menggunakan LM studio untuk bereksperimen dengan LLM lokal. Anda dapat menjalankan LLM ini di laptop kami secara offline—untuk mendownload model pertama, Anda perlu jaringan internet.

### Instalasi LM Studio

LM Studio dapat diinstal dengan mudah di Linux, Windows, dan Mac. Anda hanya perlu mengunjungi website resminya dan mengunduhnya untuk sistem operasi Anda.

### Download Model

Setelah menginstal LM Studio berhasil, langkah selanjutnya adalah mendownload model yang akan kita gunakan. Banyak model dapat didownload melalui pencarian dalam aplikasi LM Studio; di sini, kita coba mendownload model TheBloke/phi-2-GGUF.

Tunggu proses download selesai setelah Anda klik tombol download.

Mungkin kita akan mendapatkan hasil yang kurang maksimal karena TheBloke/phi-2-GGUF memiliki parameter yang kecil. Jika Anda ingin mendapatkan hasil yang maksimal, Anda bisa menggunakan model yang cukup besar seperti Llama 3–8B Instruksi.

## Kesimpulan

Begitulah cara menginstall GPT di komputer. Itu jelas, bukan? Anda sekarang dapat menggunakan GPT AI tanpa koneksi internet.

Pemilihan model harus disesuaikan dengan spesifikasi komputer Anda. Jangan paksakan menjalankan model yang memerlukan banyak sumber daya karena nantinya tidak akan bekerja dengan baik.

Selain itu, jika komputer Anda menggunakan GPU NVIDIA, Anda juga dapat menggunakan GPU CUDA.
